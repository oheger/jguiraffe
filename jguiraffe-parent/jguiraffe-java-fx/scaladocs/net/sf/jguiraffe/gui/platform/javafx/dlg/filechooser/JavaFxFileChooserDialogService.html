JavaFxFileChooserDialogService

class JavaFxFileChooserDialogService extends FileChooserDialogService

The JavaFX specific implementation of a service for displaying file selection dialogs.

This class maps the methods of the FileChooserDialogService interface to invocations of the JavaFX FileChooser and DirectoryChooser classes.
