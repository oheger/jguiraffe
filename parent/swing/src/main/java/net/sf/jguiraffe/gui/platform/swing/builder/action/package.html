This package contains Swing-specific implementations for action-related classes.

The most important class is certainly SwingActionManager, which provides a Swing-specific implementation of the ActionManager interface. SwingFormAction implements the platform-neutral concept of FormActions based on the actions available in Swing. Finally there is an implementation of the PopupMenuBuilder interface.
